## What is Prompt Engineering?
The art of writing clear instructions to get the best output from AI models like Claude and GPT.

## Key Concepts Learned

### 1. Be Specific
Bad prompt: "Write code"
Good prompt: "Write a Python function that takes a list of numbers and returns the average"

### 2. Give Context
Tell the AI who you are and what you need.
Example: "I am a Python beginner. Explain OOP classes in simple terms with examples."

### 3. Use Examples
Show the AI what output format you want.
Example: "Format the output like this: Name: X | Age: Y | Role: Z"

### 4. Role Prompting
Give the AI a role to get better responses.
Example: "You are a senior Python developer. Review this code and suggest improvements."

### 5. Chain of Thought
Ask AI to think step by step.
Example: "Solve this problem step by step, explaining each step clearly."
